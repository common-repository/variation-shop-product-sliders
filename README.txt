Variation Shop Product Sliders: Display variation products as sliders on shop/category pages.

== Description ==

Variation Shop Product Sliders: With the "variations shop product slider" plugin you could display your products variation as mini sliders on your shop page. 

This plugin will replace all products on either shop/category page with variation sliders. The center of the featured variation image will be used as its icon.

Once activated this plugin will overwrite the default woocommerce shop/category pages with the new display.

If you've found this plugin helpful, please leave a review. If you've had issues with the plugin, please leave a support request so we can give you a hand before you decide on the usefulness of it.

== Installation ==

1. Upload the "Variations Shop Product Slider" files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= What is the most amount of variation I can display per product?
You could display up to 5 variations per product
